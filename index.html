Welcome to Our Web Programming Homework

This project covers the following tasks.

JavaScript & OOJS

Interactive CRUD operations built with JavaScript, and graphical applications utilizing Object-Oriented JS classes and inheritance.

React & SPA

Component-driven UI design, state management with Hooks, and Single Page Application (SPA) routing built and compiled using Vite.

APIs & Backend

Data handling using the modern Fetch API and Axios library to communicate securely with a server-side PHP/MySQL database.
